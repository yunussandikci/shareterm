Shareterm
Developer

Easy & Secure way of sharing
terminal logs!

Minimalistc, open source online terminal log sharing tool where the server has zero knowledge of pasted data. It uses client-side encryption with AES-256 Bits.
Just pipe the shareterm and then it will return a shareble link.
